A modern, fully responsive personal portfolio website showcasing my projects, skills, and experience as a full-stack developer.

## ✨ Features

### 🎨 Design & UX
- ✅ Modern dark theme with gradient effects
- ✅ Fully responsive (mobile, tablet, desktop)
- ✅ Smooth scroll animations
- ✅ Mobile hamburger menu
- ✅ Interactive contact form

### 📱 Responsive Breakpoints
| Device | Breakpoint | Layout |
|--------|------------|--------|
| Mobile | < 640px | 1 column |
| Tablet | 640px - 1024px | 2 columns |
| Desktop | > 1024px | 4 columns |

### 📄 Sections
- **Hero Section** - Introduction with CTA buttons
- **Projects** - Showcase of my best work
- **Skills** - Technical skills with progress bars
- **About** - Personal info and statistics
- **Contact** - Contact form and social links
- **Footer** - Social media and copyright
